Temperature Explained

Low (0.1-0.5): Predictable, factual responses. Riddles are straightforward with clear logic. Chat responses are direct and helpful.
Medium (0.6-0.9): Balanced creativity. Riddles have layers and metaphors. Chat provides atmospheric guidance with some flair.
High (1.0-1.5): Maximum creativity and unpredictability. Riddles become poetic, cryptic, and surprising. Chat responses are more elaborate and imaginative.

Top-P (Nucleus Sampling)

Low (0.5-0.7): Focused responses. AI considers only the most likely word choices, resulting in coherent but less varied riddles and chat.
Medium (0.8-0.9): Balanced diversity. Good mix of predictability and creativity for engaging hunts.
High (1.0): Maximum diversity. AI considers all possible words, leading to more unexpected and varied responses, but potentially less coherent.

About This Project

Domain: Interactive Gaming & Education

API: Groq (LLaMA 3.3 70B)

Framework: Python Flask + SQLite

Problem Solved: AI generates unique, themed treasure hunt riddles with an interactive chatbot guide that provides contextual hints without revealing answers.

Data Flow: User Input → Flask API → Groq AI → JSON Response → SQLite DB → Frontend

Memory: Session-based chat history stored in SQLite database

Tech Stack
Frontend: HTML5 · CSS3 · Vanilla JS
Backend: Python 3.11 · Flask · Flask-CORS
Database: SQLite3 (players, hunts, chat_messages, clue_attempts)
AI: Groq Python SDK · llama-3.3-70b-versatile
Deployment: Localhost / Any Python host (Render, Railway, etc.)
